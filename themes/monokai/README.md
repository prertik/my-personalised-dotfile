Contents
========

Monokai color theme for Emacs. **Requires Emacs 24**.




Installation
============

This package is available from [Marmalade](http://marmalade-repo.org/). Refer
to its documentation to learn how to use Marmalade and `package.el` with your
version of Emacs.
